Background

Imagine yourself looking for a new hangout, as a tourist or a local.

How long will it take you to describe what you're looking for in a few words? Now, how long will it take you to find the place you just described? Why is it that simple to describe the OUTLINES of a hangout - style, features and vibe, yet so complicated to actually find it?

You need a search engine which is

SIMPLE | ACCURATE | SMART

Our Solution

OUTLINES is a tag-based search engine for local hangouts.
Our engine creates tags for places by analyzing millions of online user reviews across the web.
These tags reflect the sentiments, experiences and main features that other customers used in order to describe the places they've visited.
